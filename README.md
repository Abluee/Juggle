# Juggle
## 项目介绍
Juggle是一个微服务编排的产品，

## 什么时候需要Juggle
1.适合已有基础服务能力，通过Juggle编排接口快速搭建一个新产品。
2.适合做前端的适配层（即BFF），可以用Juggle替代常见的通过Nodejs来实现BFF层的能力。
3.适合需要面向私有化或大量定制开发的产品，通过Juggle编排定制化接口，避免对标准代码的污染。
